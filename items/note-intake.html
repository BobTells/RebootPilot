The Markdown Files

Generate the two prompts you need to bootstrap a markdown-based Claude system — CLAUDE.md vault, a Claude Skill, or both. The bootstrap prompt opens with a grill-me interview so the model locks your spec (especially trigger conditions and hot vs cold corpus) before it writes a single file.

Difficulty
2 / 5

Easy if you let grill-me run. Hard if you skip it and try to back-fit the spec later.

Core tools
  • Claude Code or Claude Desktop
  • grill-me skill (github.com/mattpocock/skills)
  • A markdown editor
Optional
  • Existing CLAUDE.md to extend
  • Existing .claude/skills/ folder
  • A repo to commit the system into
What it does

Two copy-ready prompts: a bootstrap prompt that grills you and writes the markdown system, plus an operational prompt for sessions after setup.
